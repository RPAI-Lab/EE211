## 任务目标

1. 从起点移动到红绿灯处，如果检测到红灯则停止，变为绿灯则继续前行
2. 从红绿灯处移动到A区域
3. 停在A区域并完成方块的抓取
4. 导航到B区域
5. 停在B区域并卸下方块
6. 导航到停车场区域（即图上标示“P”的区域）
7. 离开停车场区域，导航回到起点
8. 注意回到起点的路途中会放置“stop”标识，如果看到此标识需停止前行，等移开标识后再继续移动
